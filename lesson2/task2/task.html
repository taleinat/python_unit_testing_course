Most assertions check whether running some code yields the expected result. Usually, this means acquiring one or more values and comparing them to expected values. For this the various comparison assertions are very useful.

We've already mentioned assertEqual as well as assertTrue and assertFalse, all of which are comparison assertions. Another such assertion method is assertNotEqual(a, b), which is the opposite of assertEqual. Similarly, most assertion methods have matching negated methods which check the opposite condition.

Some additional comparison assertions are:

  • assertIs(a, b): check whether a is b
  • assertIsNot(a, b)
  • assertIsNone(a): check whether a is None
  • assertIsNotNone(a)
  • assertGreater(a, b): check whether a > b
  • assertGreaterEqual(a, b): check whether a >= b
  • assertLess(a, b): check whether a < b
  • assertLessEqual(a, b): check whether a <= b
