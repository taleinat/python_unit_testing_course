This course will teach how to write unit tests for Python functions and classes. More specifically, it teaches how to use unit testing to make developing Python code easier and more enjoyable!

We'll start with an example, to see what a simple unit test looks like.

This will also show that unit tests are a very explicit documentation of what a piece of code does: even without seeing its code, it is obvious what the function should do!

Your objective: Make the test pass by implementing the hello_world() function.

To run the tests:

  1. first, click the green "run" button at the top-left of the code window once.
  2. A Unittest in hello_wold run configuration will now appear on the top-right of the PyCharm window. From now on, you can click the green "run" button next to it to just run the unit tests.
